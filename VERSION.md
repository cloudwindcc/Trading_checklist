## 🤝 协作开发

### 开发环境设置
```bash
cd trading-checklist-v0.2
npm install          # 安装依赖
npm run dev          # 启动开发服务器
```

### 分支策略
- `main`：稳定版本
- `develop`：开发分支
- `feature/*`：功能分支

### 提交规范
```
feat: 添加新功能
fix: 修复bug
docs: 文档更新
style: 样式调整
refactor: 代码重构
test: 测试相关
```
